# jobss
Job-seeking application that I made for OBSS 2023 Internship. <br/>
<h1>Technology stack:</h1>
<h3>Frontend:</h3>
<ul>
  <li>TypeScript</li>
  <li>NextJS</li>
  <li>Tailwind CSS</li>
  <li>ShadcnUI</li>
  <li>MaterialUI</li>
</ul>
<h3>Backend:</h3>
<ul>
  <li>Spring Boot</li>
  <li>Spring Cloud Gateway,</li>
  <li>Eureka</li>
  <li>Apache Kafka</li>
  <li>Spring Security</li>
  <li>PostgreSQL</li>
</ul>
